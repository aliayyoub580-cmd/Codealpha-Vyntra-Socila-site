# Vyntra Social Source Notes

The shipped application is intentionally simple and framework-free:

- `public/` contains the responsive Bootstrap frontend.
- `server/` contains the Express API and Supabase integration.
- `supabase/` contains SQL for tables, row-level security, storage buckets, and helper triggers.

This keeps the project aligned with the requested HTML, CSS, Bootstrap, JavaScript, Node.js, Express.js, and Supabase stack.
